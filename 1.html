📐 Задача с параметром

Условие задачи №18
Найдите все значения параметра \(a\), при каждом из которых уравнение
$$(\operatorname{tg} x + 6)^2 - (a^2 + 2a + 8)(\operatorname{tg} x + 6) + a^2(2a + 8) = 0$$
имеет на отрезке \(\left[0; \frac{3\pi}{2}\right]\) ровно два решения.
Начало Шаг 1 из 6 Ответ
1 Замена переменной

Введём замену для упрощения уравнения:

Замена $$t = \operatorname{tg} x + 6$$

Получаем квадратное уравнение относительно \(t\):

$$t^2 - (a^2 + 2a + 8)t + a^2(2a + 8) = 0$$
$$D = (a^2 + 2a + 8)^2 - 4a^2(2a + 8)$$ $$D = a^4 + 4a^3 + 20a^2 + 32a + 64 - 8a^3 - 32a^2$$ $$D = a^4 - 4a^3 - 12a^2 + 32a + 64$$ $$D = (a^2 - 2a - 8)^2$$
2 Нахождение корней

Находим корни квадратного уравнения:

Первый корень
$$t_1 = \frac{(a^2 + 2a + 8) + (a^2 - 2a - 8)}{2} = a^2$$
Второй корень
$$t_2 = \frac{(a^2 + 2a + 8) - (a^2 - 2a - 8)}{2} = 2a + 8$$
3 Возврат к переменной x

Возвращаемся к исходной переменной:

$$\operatorname{tg} x + 6 = a^2 \Rightarrow \operatorname{tg} x = a^2 - 6$$ $$\operatorname{tg} x + 6 = 2a + 8 \Rightarrow \operatorname{tg} x = 2a + 2$$

Получили два уравнения:

Уравнение 1: \(\operatorname{tg} x = a^2 - 6\)
Уравнение 2: \(\operatorname{tg} x = 2a + 2\)
4 Анализ количества решений

Исследуем поведение функции \(\operatorname{tg} x\) на отрезке \(\left[0; \frac{3\pi}{2}\right]\):

Ключевой факт На отрезке \(\left[0; \frac{3\pi}{2}\right]\):
  • При \(c \geq 0\): 2 решения
  • При \(c < 0\): 1 решение

Обозначим \(N(c)\) — количество решений уравнения \(\operatorname{tg} x = c\):

$$N(c) = \begin{cases} 2, & c \geq 0 \\ 1, & c < 0 \end{cases}$$
5 Составление условия

По условию задачи общее количество решений должно быть равно 2:

$$N(a^2 - 6) + N(2a + 2) = 2$$

Возможные случаи:

Случай A

2 + 0 = 2 (невозможно, минимум 1)

Случай B

1 + 1 = 2 ✓

Случай C

Кратный корень (D = 0)

6 Решение системы

Случай B: Оба выражения отрицательны

$$\begin{cases} a^2 - 6 < 0 \\ 2a + 2 < 0 \end{cases}$$

Решаем систему:

$$\begin{cases} a^2 < 6 \Rightarrow -\sqrt{6} < a < \sqrt{6} \\ 2a < -2 \Rightarrow a < -1 \end{cases}$$
Пересечение: \(-\sqrt{6} < a < -1\)

Случай C: Кратный корень

При \(D = 0\):

$$a^2 - 2a - 8 = 0 \Rightarrow (a-4)(a+2) = 0$$ $$a = 4 \quad \text{или} \quad a = -2$$

Проверка:

  • При \(a = 4\): \(\operatorname{tg} x = 10 \geq 0\) ⇒ 2 решения ✓
  • При \(a = -2\): \(\operatorname{tg} x = -2 < 0\) ⇒ 1 решение ✗

Проверка граничных точек

  • \(a = -\sqrt{6}\): 2 + 1 = 3 решения ✗
  • \(a = -1\): 1 + 2 = 3 решения ✗
✓ Ответ
$$a \in (-\sqrt{6}; -1) \cup \{4\}$$
или
\(-\sqrt{6} < a < -1\) или \(a = 4\)
